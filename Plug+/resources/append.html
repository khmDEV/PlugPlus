Link Expansion Links in chat and the page description will automatically get expanded and previewed. (Only pictures and youtube)
Afk Message Enter a message that should be printed if you are afk. Empty will disable printing afk messages. (Status must be afk)
Auto AFK This will set your status to afk if you are not active on the page for X minutes. This will set it to available if you return. 0 disables this feature. (Must refresh to update changes)
Global Notifications Disable this to disable all notifications completely.
Require Blur This will require that plug.dj is not the active window to display notifications.
Chat Level
Choose what chats you would like to be notified of. Disable all options to disable. Using the all option will override the others.
User Level
Choose what degree of notification you would like for users leaving, joining, and voting.
Song Update
Choose what songs you would like to be notified of.
DJ Update
Select what DJs you would like to be notified of.
AutoWoot Delay Choose a number in seconds to wait before wooting. (0-90)
